This was a very intresting project which I learned alot from. Not only were we supossed to replicate the program Powerpoint onto Scratch we also had to come up with a presentation. He also wanted it to be able to run multiple School-like programs like Microsoft Word ect. I really think this project was the highlight of the Scratch unit!
